# MC Inventory

MC Inventory is a SaaS platform designed to centralize and automate AWS infrastructure inventory management in hybrid and multicloud environments.

## 🚀 Features

- Real-time AWS inventory sync
- EC2 instance detection
- RDS database detection
- ECS cluster detection
- CloudFront distribution detection
- Automatic refresh
- Modern dashboard UI
- Status visualization (Running / Stopped)

## 🛠 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- AWS SDK v3
- TailwindCSS
- REST API (Server-side integration)
